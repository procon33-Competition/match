### やってることメモ

* 音を時間を持つ画像にする(fftやらなんやらで)
* この画像のピーク、もしくは特徴点を見つける
* その点をまわりの点とペアリングする
* このペアリングした情報をハッシュテテーブに保存

* マッチングするときは上の手法して全探索 O(n)
* 実際の計算量はハッシュの数*問題音声の数(未確認)
